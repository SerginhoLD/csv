# CSV Parser (RFC 4180)

## Installation
```bash
composer require serginhold/csv
```

## Example

CSV:
```csv
Pot,Club,Country
1,"PFC CSKA Moskva",RUS
2,Bayer 04 Leverkusen,GER
3,"Tottenham
Hotspur FC",ENG
4,Monaco,FRA
```

Code:
```php
$csv = new \SerginhoLD\Csv\Parser();
$arCsv = [];

foreach ($csv->parseFile(__DIR__ . '/sample.utf8.csv') as $row)
{
    $arCsv[] = $row;
}

echo '<pre>' . print_r($arCsv, true) . '</pre>';
```
